# Project Overview

This project focuses on analyzing customer churn data to understand why customers leave and which factors contribute most to churn.
The interactive dashboard is designed to provide clear insights into customer behavior and support data-driven decision-making for retention strategies.

# Dataset
7,043 customer records
23 columns including demographics, services, contract details, billing information, and churn status

# Key Metrics
Total Customers: 7,043
Churned Customers: 1,869
Churn Rate: 26.54%
Retention Rate: 73.46%
Average Tenure: 25.35 months
Average Monthly Charges: $25

# Analysis & Insights
Churn patterns across age groups, gender, and tenure
Contract type and payment method impact on churn
Service-based churn analysis (Internet, Tech Support, Security, etc.)
Identification of high-risk customer segments
Revenue impact of customer churn

# Dashboard Features
Interactive slicers for dynamic analysis
Decomposition Tree to identify key churn drivers
Clear KPI cards and trend-based visuals
User-friendly layout focused on insight discovery

# Tools Used
Power BI
DAX

# Purpose

The goal of this project is to demonstrate practical data analytics and visualization skills by transforming raw customer data into meaningful business insights through an interactive dashboard.
